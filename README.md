# Project 6: AI Review Responder

## Live Demo
**[ai-review-responder-dfmu.onrender.com](https://ai-review-responder-dfmu.onrender.com)**

## Service Demonstrated
**Reputation Management Automation** — clients get AI-drafted responses to Google,
Yelp, and other platform reviews. Saves 30–60 minutes per week for restaurant
owners and service businesses, while keeping tone consistent and professional.

## Goal
Build a demo where a user pastes a customer review and receives 2–3 professionally
drafted response options — showing exactly the output a real client receives as
part of an automated reputation management workflow.

## Stack
- Python / Flask
- OpenRouter API (free tier)
- Render (free tier deployment)

## Run Locally
```bash
pip install -r requirements.txt
OPENROUTER_API_KEY=your_key python app.py
```
